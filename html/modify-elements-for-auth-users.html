Modifying HTML elements is suitable for making minor changes within existing elements, such as updating text, changing styles, or tweaking attributes.

2. Injecting New HTML Code

When to Use Injecting New HTML:

Injecting new HTML code is ideal for replacing entire sections, integrating static content, or embedding third-party elements.

Creating a Container Element:

Create an HTML container element to insert new HTML content.

<!-- Create a container for injecting new HTML -->

<div id="htmlContainer"></div>

Using JavaScript to Insert HTML:

NOTE: You MUST use backticks(``) when creating your html string.

Inject HTML code into the container using the innerHTML property.

// Your HTML string

const htmlString = `<h1>Hello, <em>World!</em></h1>` ;

// Get the container element

const container = document.getElementById('htmlContainer');

// Insert the HTML string into the container

container.innerHTML = htmlString;
